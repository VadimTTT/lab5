## Ответы на вопросы

### Задание 2
1. Расширения: C `.c`, C++ `.cpp/.cc`, Header `.h`, Objective-C `.m`, Objective-C++ `.mm`, Object file `.o`.
2. Вывод: `NSLog`, `printf`, `fprintf`.

### Задание 3
1. Метод количества в `NSArray`: `count`.
2. Правильный вариант: **B** и **D** (оба создают mutable массив).
3. Добавление в конец: `addObject:`.
4. При выходе за границы — исключение `NSRangeException`.
5. Последний элемент: `lastObject`.

### Задание 4
1. Swift REPL — интерактивный режим Swift для быстрых проверок кода.
2. Playground — среда для экспериментов и обучения, прототипирования логики.
3. Компиляция в bash: `swiftc main.swift -o app && ./app`.

### Задание 6
1) Режимы Xcode: редактор проекта, playground, simulator/debugger, interface builder.  
2) Swift безопаснее (опционалы, type safety), современный синтаксис, ARC встроен.  
3) Отладка: breakpoints, LLDB-консоль, step over/into/out, watch variables.  
4) Операторы: арифметические, сравнения, логические, присваивания, диапазоны, `nil-coalescing`.  
5) Коллекции: `Array`, `Set`, `Dictionary` (упорядоченность, уникальность, ключ-значение).  
6) Типы проектов: iOS/macOS/watchOS/tvOS App, Framework, Command Line Tool, Package.

### Задание 7
1) Пустой массив `Int`: `var a: [Int] = []`.  
2) Для `let` нельзя менять размер массива.  
3) Добавление в конец: `append`.  
4) Все элементы массива должны быть одного типа.  
5) «Универсальная коллекция» — может хранить набор значений одного типа и поддерживает богатые операции обработки.  
6) Перебор с индексом: `enumerated()`.
