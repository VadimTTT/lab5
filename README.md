# Lab 5 — iOS / Objective-C / Swift

## Project Name
Laboratory work 5 (Xcode, Objective‑C, Swift console applications)

## Description
Repository contains tasks 2, 3, 4, 6, and 7 for variant 22.
Author: **Тюшко Вадим**, group 12.

Implemented directories:
- `task2` — Objective‑C console app (basketball penalty time by penalty type).
- `task3` — Objective‑C arrays (`NSArray` sorting by string length).
- `task4` — Swift rating calculator for university admission.
- `task6` — Swift dictionary operations for software/year and software/license.
- `task7` — Swift MVC app with `ArraySplitter` (optional array split into even/odd).

## Installation
1. Clone repo.
2. Open needed task in Xcode **or** compile console files from terminal (`swiftc` for Swift).
3. For Objective‑C tasks, create a macOS Command Line Tool project in Xcode and place files from corresponding folder.

## Usage
- Run each task as console app.
- Testing protocols are in the lab report.
